# GEE_rf_landcover_classification
The repo contains the script used for land cover classification using Landsat 5 and Landsat 8 images
The script is part of the project which analyzed the land cover change between 1992 and 2020
The files are:
  Features used for training a random forest classifier
  Script used for land cover classification using random forest
  Script used for surface water detection using Sentinel-1 SAR images
